Features & how it works

The whole clip pipeline, explained

AstroClipz automates the repetitive work between a livestream moment and a posted clip. Here's what happens at each stage, and the features that power it.

01

Continuous capture

AstroClipz keeps a rolling buffer of the live stream — always recording the recent past so a moment can be clipped the instant it happens, without recording hours of footage you'll never use. Only a short recent window is kept on disk at any time, and it's continuously cleaned up.

02

Automatic moment detection

Instead of watching for hours, AstroClipz reads the room. It measures live chat velocity and engagement in real time and, when a moment spikes past a threshold tuned per streamer, marks that window for clipping. A quick calibration step learns each channel's normal chat baseline so busy chats and quiet ones are both handled sensibly.

03

Vertical formatting

Landscape gameplay doesn't belong on a vertical feed. AstroClipz scales the footage into a 9:16 canvas over a clean black or blurred-fill background, and burns in an editable headline placed exactly where you want it — the packaging that makes a clip feel native to TikTok, Shorts and Reels.

04

Word-by-word captions

Speech is transcribed automatically and rendered as animated, word-by-word captions. The transcript is fully editable, so you can fix a word or reposition the caption block, then the captions are burned cleanly into the final render — the retention feature short-form audiences expect.

05

Branding your way

Each clip can carry a platform lockup with the source handle, a custom campaign watermark you upload, or no branding at all. Position, size and opacity are set per clip, so a client campaign and an unbranded post can come off the same pipeline.

06

Human review & approval

Every finished clip lands in a private review queue in a mobile-friendly app. You watch it, fine-tune the trim, edit the caption and hashtags, and either approve or reject. Nothing is ever posted automatically — a person signs off on every single clip.

07

Publish through official APIs

Approved clips upload straight to connected accounts through each platform's official posting API — starting with the TikTok Content Posting API — with the caption and hashtags you set. Accounts are connected only through the platform's own secure authorisation flow and can be disconnected at any time.
